# Métricas por criterio — Estructura de evaluación

Este documento define las métricas específicas que el evaluador debe comprobar para determinar el nivel y calificación de cada criterio, según la rúbrica.

---

## Cognitivo (máx 40 puntos)

**Pregunta detonadora 1:** "¿Para qué me sirve saber las diferencias entre relato, resumen y reseña?"

**Pregunta detonadora 2:** "¿Cómo las puedo aplicar en mi contexto de vida?"

### Métrica Cognitiva: Aportaciones y contenido

| Métrica | ¿Cumple? | Nivel resultante |
|---------|----------|-----------------|
| **5 aportaciones totales** (1-5 partes) | | |
| Aportación 1: responde PD1 con fundamentos claros y concisos | Sí | Experto (40) |
| Aportación 2: responde PD2 con fundamentos claros y concisos | Sí | Experto (40) |
| **2 aportaciones** comentan compañeros (opinión fundamentada) | Sí | Experto (40) |
| **1 aportación** conclusión sobre temática | Sí | Experto (40) |
| | | |
| **4 aportaciones totales** | | |
| Aportación 1: responde PD1 de manera general | Sí | Capacitado (34) |
| Aportación 2: responde PD2 con fundamentos generales | Sí | Capacitado (34) |
| **2 aportaciones** opinión general compañeros | Sí | Capacitado (34) |
| **1 aportación** conclusión | Sí | Capacitado (34) |
| | | |
| **3 aportaciones totales** | | |
| Aportación 1: responde PD1 superficialmente | Sí | Aceptable (32) |
| Aportación 2: responde PD2 superficialmente | Sí | Aceptable (32) |
| **1 aportación** opinión (un compañero, se dificulta fundamentar) | Sí | Aceptable (32) |
| Conclusión general incluida | Sí | Aceptable (32) |
| | | |
| **3 aportaciones** | | |
| Aportación 1: responde PD1 vagamente | Sí | Aprendiz (28) |
| Aportación 2: responde PD2 vagamente | Sí | Aprendiz (28) |
| Opinión vagua sin fundamento (un compañero) | Sí | Aprendiz (28) |
| Conclusión vaga | Sí | Aprendiz (28) |
| | | |
| **1 aportación sola** | | |
| Responde vagamente a PD1 y PD2 en una sola aportación | Sí | Requiere apoyo (24) |
| Comenta vagamente a un compañero | Sí | Requiere apoyo (24) |
| Comentario final sin conclusión clara | Sí | Requiere apoyo (24) |
| | | |
| **No participa o no cumple mínimos** | Sí | No evaluable (0) |

**Lógica de asignación:** Evaluar de arriba hacia abajo; si cumple todas las métricas de **Experto**, se asigna level Experto (40). Si no, pasar a **Capacitado**, etc.

## Actitudinal (máx 15 puntos)

**Criterio:** Cumplimiento de instrucciones para participar.

| Porcentaje de cumplimiento | Puntos |
|---------------------------|--------|
| **100%** — Atiende todas las instrucciones | 15 |
| **90%** — Atiende 90% de instrucciones | 14 |
| **80%** — Atiende 80% de instrucciones | 12 |
| **70%** — Atiende 70% de instrucciones | 10 |
| **≤60%** — Atiende 60% o menos | 9 |
| **No evaluable** | 0 |

**Instrucciones a verificar (del documento M2_S4_foro_integracion_instrucciones.md):**
- [ ] Usa creatividad y originalidad
- [ ] Cita fuentes si usa recursos
- [ ] Intervenciones generan diversidad
- [ ] Evita subir documentos (usa enlaces)
- [ ] Respeta netiqueta
- [ ] Opiniones breves (1-3 párrafos)
- [ ] Redacción previa (sin errores plataforma)

**Lógica:** Contar cumplimientos y calcular porcentaje.

## Flujo de evaluación esperado

1. **Formulario dinámico:** mostrar métricas por criterio → checkboxes/radio por métrica.
2. **Lógica:** comparar respuestas → determinar nivel automáticamente → asignar puntos.
3. **Cálculo:** sumar puntos por criterio + deducción originalidad = total.
4. **Retroalimentación:** construir dinámicamente según nivel de cada criterio + plantillas predefinidas.
